# Flagship — 30-Minute Demo Store Setup Guide

Follow these steps in your Shopify admin and the homepage transforms from
placeholders into a full premium storefront. Themes provide the design;
products, menus and pages are store content that only the admin can create.

## 1. Products (15 min) — biggest visual impact

**Products → Add product**, create 6–8 fashion items. For each:

- Title, price, and 2–4 photos (free sources: unsplash.com, pexels.com)
- On at least 3 products: add an option named **Color** with values like
  Sand / Olive / Black → this activates color swatches on cards and the
  product page
- On 2 products: set **Compare-at price** higher than price → activates the
  "-20%" sale badges
- On 1 product: Inventory → track quantity → set to **2** → activates the
  "Hurry — only 2 left" alert
- Set product **Vendor** to your brand name variations if you want vendor
  lines on cards

## 2. Collections (5 min)

**Products → Collections → Create collection** — make three, e.g.
*Outerwear*, *Accessories*, *Knitwear* (manual type; add your products).
Also confirm the automatic **All** collection exists.

## 3. Navigation (5 min) — activates the mega menu

**Online Store → Navigation → Main menu**:

- Add items: *Shop all* → All collection; *Outerwear* → collection;
  *Accessories* → collection; *Journal* → blog; *About* → page; *Contact* → page
- Drag items **underneath** another item to nest them — a nested item under
  a parent creates the dropdown; two levels of nesting creates the full
  mega menu panel

**Footer menu**: add FAQ, Contact, and your policy pages
(policies are generated in Settings → Policies).

## 4. Pages (5 min)

**Online Store → Pages**:

- *About* — a paragraph about the brand
- *Contact* — assign theme template **contact** in the right sidebar →
  renders the styled contact form
- *FAQ* — optional

## 5. Blog

**Online Store → Blog posts** — add 2–3 posts with cover images to fill
the "Stories from the atelier" section (it shows styled samples until then).

## 6. Theme editor wiring (5 min)

Open the theme editor and click each homepage section:

- **Best sellers** → pick your All collection
- **Shop by category** → pick your three collections
- **Featured product** (Rich product hero) → pick your flagship product
- **Slideshow / banners / tiles** → upload your own photos (they show
  bundled demo photography until you do)
- **Journal** → pick your blog

## 7. Filters & wishlist (5 min)

- Install the free **Shopify Search & Discovery** app → Filters → add
  Color, Size, Price, Availability, Product type → Save. The left filter
  sidebar on collections and search fills automatically.
- **Online Store → Pages → Add page** → title "Wishlist" → Theme template
  **wishlist** → Save. Then paste the page link into
  Customize → Header → "Wishlist page link" to show the heart icon.

## 8. Optional finishing touches

- Settings → Markets: add countries → currency selector goes live
- Settings → Customer accounts: choose account type (theme includes full
  branded account pages for the Legacy option)
- Theme settings → try the three presets: Lumen (warm light),
  Mode (black & white editorial), Vanta (dark)
